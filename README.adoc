== Pré-requis pour construire les exemples

* Docker Engine 19.03.0+
* Docker compose

Optionnel

* GNU Make

== Génération des slides


[source,shell]
----
docker compose run build-slides
# Ou avec GNU Make
make 01-slides
----

== Génération du Pdf

[source,shell]
----
docker compose run build-pdf
# Ou avec GNU Make
make 01-pdf
----

== Génération du E-book

[source,shell]
----
docker compose run build-epub
# Ou avec GNU Make
make 01-epub
----
